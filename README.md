# Elevate Labs - Data Analyst Internship Task 1

## 📌 Task Name
Data Cleaning and Preprocessing

## 🎯 Objective
To clean and prepare a raw dataset by handling missing values, duplicates, inconsistent text formats, date formats, column names, and data types.

## 🛠 Tools Used
- Python
- Pandas
- NumPy

## 📂 Dataset Used
Netflix Movies and TV Shows Dataset

## 🚀 Steps Performed
1. Imported the dataset
2. Checked basic information
3. Identified and handled missing values
4. Removed duplicate rows
5. Standardized text values
6. Renamed column headers
7. Converted date formats
8. Checked and fixed data types
9. Saved the cleaned dataset

## 📁 Files Included
- Original dataset
- Cleaned dataset
- Python code / Notebook
- README.md

## ✅ Output
A cleaned and structured dataset ready for analysis.

## 📚 Learning Outcome
This task helped in understanding how to clean real-world datasets before analysis or visualization.
